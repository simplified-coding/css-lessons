# Μάθημα 8ο: Λίστες και Πίνακες

<h1 style="text-align: center;"> Μάθημα 8 </h1>

<h2 style="text-align: center;">  Λίστες και Πίνακες </h2>

---

<h2 style="text-align: center;">  Λίστες </h2>

Με τη CSS μπορούμε να προσαρμόσουμε μια λίστα με τρόπους όπως:

* ```list-style-type```: Να αλλάξουμε τον τύπο των σημείων (bullets ή αριθμοί) που χρησιμοποιούνται
    - ```circle```: Κύκλοι
    - ```square```: Τετράγωνα
    - ```upper-roman```: Κεφαλαίοι αριθμοί (Λατινικά)
    - ```lower-roman```: Μικροί αριθμοί (Λατινικά)
* ```list-style-image```: Να ορίσουμε μια εικόνα ως σημείο
    - ```url('image.png')```: Η εικόνα που θα χρησιμοποιηθεί
* ```list style-position```: Να ορίσουμε τη θέση των σημείων
    - ```inside```: Μέσα στο κείμενο
    - ```outside```: Εκτός του κειμένου

Παράδειγμα:

```css
ul {
    list-style-type: square;
    list-style-image: url('image.png');
    list-style-position: inside;
}
```

---

<h2 style="text-align: center;">  Πίνακες </h2>

Με τη CSS μπορούμε να προσαρμόσουμε έναν πίνακα με τρόπους όπως:

* ```border-collapse```: Να ενώσουμε τα περιθώρια των κελιών
    - ```collapse```: Να ενώσουμε τα περιθώρια
    - ```separate```: Να διαχωρίσουμε τα περιθώρια
* ```border-spacing```: Να ορίσουμε το περιθώριο μεταξύ των κελιών
    - ```10px```: Το περιθώριο θα είναι 10 pixels
* ```border```: Να ορίσουμε το περιθώριο των κελιών
    - ```1px solid black```: Το περιθώριο θα είναι 1 pixel, μαύρο και συνεχές

Παράδειγμα:

```css
table {
    border-collapse: collapse;
    border-spacing: 10px;
    border: 1px solid black;
}
```
