<h2 style="text-align: center;"> Χρώμα </h2>

Είδη χρωμάτων στην CSS:
* Χρώμα υποβάθρου (background-color)
* Χρώμα κειμένου (color)
* Χρώμα περιγράμματος (border)

---

<h2 style="text-align: center;"> Χρώμα υποβάθρου </h2>

Στην CSS μπορούμε να ορίσουμε ένα χρώμα ως υπόβαθρο για έναν τίτλο, παράγραφο και άλλα.

Για παράδειγμα, αν θέλουμε να ορίσουμε το χρώμα υποβάθρου του τίτλου σε κόκκινο, θα γράψουμε:

```css
h1 {
  background-color: red;
}
```

---

<h2 style="text-align: center;"> Χρώμα κειμένου </h2>

Επιπλέον, μπορούμε να ορίσουμε και χρώμα κειμένου.

Παράδειγμα:

```css
h1 {
  color: blue;
}
```

---

<h2 style="text-align: center;"> Χρώμα περιγράμματος </h2>

Στην CSS μπορούμε να ορίσουμε ένα χρώμα περιγράμματος για έναν τίτλο, παράγραφο και άλλα.
Αυτή η διαδικασία είναι πιο περίπλοκη σε σχέση με τα προηγούμενα.

Παράδειγμα:

```css
h1 {
  border: 2px solid green;
}
```

Βλέπουμε ότι αντί να έχουμε μόνο το χρώμα, υπάρχει και η λέξη `solid` και ο αριθμός `2px`.
Αυτά είναι τα είδη του περιγράμματος και το πάχος του.
Θα δούμε περισσότερα είδη σε μελλοντικό μάθημα.
