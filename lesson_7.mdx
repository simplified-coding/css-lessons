# Μάθημα 7ο: Κείμενο, Γραμματοσειρά και Σύνδεσμοι

<h1 style="text-align: center;"> Μάθημα 7 </h1>

<h2 style="text-align: center;"> Κείμενο, Γραμματοσειρά και Σύνδεσμοι </h2>

---

<h2 style="text-align: center;"> Κείμενο </h2>

Η CSS έχει πολλούς τρόπους για να προσαρμόσουμε ένα κείμενο.
Μερικούς από αυτούς είναι:

* Χρώμα (Το οποίο το έχουμε δει σε προηγούμενο μάθημα)
* Ευθυγράμμιση
* Απόσταση
* Σκιά

<h3 style="text-align: center;"> Ευθυγράμμιση </h3>

Στην CSS υπάρχουν αρκετο τρόποι ευθυγράμμισης κειμένου, αλλά εμείς θα δούμε τον πιο βασικό.
Για να ευθυγραμμίσουμε το κείμενο στην CSS χρησιμοποιούμε την ιδιότητα ```text-align``` και τις τιμές ```left```, ```right``` και ```center```.
Με την τιμή ```left``` το κείμενο ευθυγραμμίζεται στα αριστερά, με την τιμή ```right``` στα δεξιά και με την τιμή ```center``` στο κέντρο.

Παράδειγμα:

```css
p {
    text-align: center;
}
```

<h3 style="text-align: center;"> Απόσταση </h3>

Στην CSS υπάρχουν 4 τρόποι απόστασης κειμένου. Αυτοί είναι:

* ```letter-spacing```: Απόσταση μεταξύ των γραμμάτων
* ```word-spacing```: Απόσταση μεταξύ των λέξεων
* ```line-height```: Απόσταση μεταξύ των γραμμών
* ```text-indent```: Απόσταση της πρώτης γραμμής

Παράδειγμα: 

```css
p {
    letter-spacing: 2px;
    word-spacing: 5px;
    line-height: 1.5;
    text-indent: 10px;
}
```

<h3 style="text-align: center;"> Σκιά </h3>

Μπορούμε να προσθέσουμε σκιά στο κείμενο μας με την ιδιότητα ```text-shadow```.
Η ιδιότητα αυτή έχει 3 τιμές. Αυτές είναι:

* Οριζόντια απόσταση σκιάς
* Κάθετη απόσταση σκιάς
* Χρώμα σκιάς

Παράδειγμα:

```css
p {
    text-shadow: 2px 2px #000000;
}
```

Βλέπουμε πως η 1η τιμή είναι η οριζόντια απόσταση, η 2η η κάθετη απόσταση και η 3η το χρώμα.

---

<h2 style="text-align: center;"> Γραμματοσειρά </h2>

Στην CSS μπορούμε να προσαρμόσουμε την γραμματοσειρά του κειμένου μας.
Για να το κάνουμε αυτό χρησιμοποιούμε την ιδιότητα ```font-family```.
Η ιδιότητα αυτή έχει 2 τιμές. Αυτές είναι:

* Όνομα γραμματοσειράς
* Τύπος γραμματοσειράς

Μερικές από τις γραμματοσειρές που μπορούμε να χρησιμοποιήσουμε είναι:

* ```Arial```
* ```Verdana```
* ```Times New Roman```
* ```Courier New```
* ```Georgia```
* ```Comic Sans MS```

Ο τύπος γραμματοσειράς μπορεί να είναι ```serif``` ή ```sans-serif```.

Παράδειγμα:

```css
p {
    font-family: Arial, sans-serif;
}
```

Βλέπουμε πως η 1η τιμή είναι το όνομα της γραμματοσειράς και η 2η ο τύπος.

---

<h2 style="text-align: center;"> Σύνδεσμοι </h2>

Στην CSS μπορούμε να προσαρμόσουμε τους συνδέσμους μας.
Για να το κάνουμε αυτό χρησιμοποιούμε την ιδιότητα ```a```.
Η ιδιότητα αυτή είναι λίγο διαφορετική, γιατί αλλάζει ανάλογα με την κατάσταση του συνδέσμου.
Αυτές είναι:

* ```a:link```:     Ο σύνδεσμος όταν δεν έχει πατηθεί
* ```a:visited```:  Ο σύνδεσμος όταν έχει πατηθεί
* ```a:hover```:    Ο σύνδεσμος όταν ο δείκτης του ποντικιού βρίσκεται πάνω του
* ```a:active```:   Ο σύνδεσμος όταν έχει πατηθεί

Αυτές οι καταστάσεις μπορούν να πάρουν ως τιμή ιδιότητες όπως:

* ```color```: Χρώμα
* ```text-decoration```: Διακόσμηση κειμένου 
    - ```none```: Καμία
    - ```underline```: Υπογράμμιση
    - ```overline```: Πάνω γραμμή
    - ```line-through```: Διαγραφή
    - ```blink```: Αναβοσβήνει
* ```background-color```: Χρώμα φόντου

Παράδειγμα:

```css
a:link {
    color: #0000FF;
    text-decoration: underline;
}
```
